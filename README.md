# esp32-logger

## Libraries

You need:
 - Adafruit DHT
 - Adafruit Unified Sensor

## Setup

Fork the repo, clone, and setup to match your setup.

You can change:

| Name | Options | Use |
| ---- | ------- | --- |
| LED_PIN | any, default **2** | Status & debug LED, built-in or connected by pin |
| LIGHT_SENSOR_PIN | any, default **D32** | Your photoresistor |
|  |  |  |
|  |  |  |
|  |  |  |
